# RPL Capstone - Wash N GO

Proyek ini adalah aplikasi Capstone untuk Rekayasa Perangkat Lunak (RPL) yang mengutamakan fiture booking.

## Persiapan (Prerequisites)
Agar aplikasi dapat berjalan secara lokal, pastikan Anda telah mengimpor database dan mengaktifkan layanan berikut:
1. **Web Server** (Apache / Nginx)
2. **Database** (MySQL)

## Instalasi API (Backend)
Silakan letakkan folder `washngo_api` pada direktori root web server Anda sesuai aplikasi yang digunakan:

### 1. Pengguna Laragon
Letakkan folder di:
`{Lokasi Instalasi}\laragon\www\`
*(Contoh: C:\laragon\www\washngo_api)*

### 2. Pengguna XAMPP
Letakkan folder di:
`{Lokasi Instalasi}\xampp\htdocs\`
*(Contoh: C:\xampp\htdocs\washngo_api)*
